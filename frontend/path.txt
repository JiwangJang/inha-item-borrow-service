할일
1. 관리자가 요청 허가, 불허가 할 수 있도록 개발 - 수정기능도 제작 V
2. 대여자가 불허가 받을 경우 불허가 사유 볼 수 있도록 하기 + 불허가했다고 대여신청서 상에서 알리는거 만들기 + 관리자 배정후에는 수정 버튼 숨기기
4. 서버에서 대여시간 검증하도록 하기(대여일시는 요청순간보다 1시간 뒤여야 하고, 반납일시는 대여일시보다 나중이어야함)

반납요청 조회할 때, 이전 요청도 가져오게 하기
아이템 아이디 같고, 대여자 아이디 같으면 이전 대여요청임

대여자가 로그인 하면 새정보가져오기
사용자가 요청했다가 빠꾸먹으면 새 요청 올려야함.

대여시
대여신청 -> 학생회 카톡 연락(보증금 납부여부도 함께) -> 대여승인 -> (사물함에 넣을수 없는 경우) 대면 대여

반납시
반납신청 -> 학생회 카톡 연락 -> 대면 후 물품상태 검사 -> 반납 승인
